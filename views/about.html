About

Jam icons is a set of icons designed for web projects, illustrations, print projects, etc. Shipped in both font & svg versions. I aimed to provide a large choice of icons for all projects : filled & outlined icons.
The icons have been made in a 64x64 grid with a 4 pixels stroke basis, all in a consistent way and tested at 18x18, 24x24, 32x32.

Do you provide a font for Jam icons?

I do provide a very basic web font with no concern about unicode position to use each font. If you need to have a more specific approach, I provided the icons in SVG.

How do I use Jam icons?

You can use Jam icons in two ways: as a font, with its css stylesheet or as independent SVGs files for all your projects: web, app, print etc. Do wathever you want .

If you use the font, you can easily use the CDN link, so you don't have to download anything: <head>
  ...
  <link rel="stylesheet" href="https://unpkg.com/jam-icons@1.0.72/css/jam-icons.min.css">
</head>
Install via npm:
  $  npm install jam-icons


Or download Jam and add it to your project with the following tag: <head>
  ...
  <link rel="stylesheet" href="path-to/css/jam-icons.min.css">
</head> *Don't forget to copy the "fonts" folder at the same level of your css folder.

To use an icon, add the master class "jam" to your tag and just click on the needed icon or prefix it with "jam-".
For example, if you need to use the "backpack" icon, do as below: <span class="jam jam-backpack"></span>

Is Jam icons free? Could I use it for comercial projects?

Jam icons is an open sourced project available on Github, under MIT licence.
You can use all the icons, font for both personal and professional projects.
A reference to this project website is appreciated . For example: <a href="http://jam-icons.com" title="Jam icons site" target="_blank">Icons by Jam icons</a>
